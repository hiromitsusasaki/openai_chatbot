# OpenAI Chatbot API

## Description

OpenAI Chatbot API is a RESTful API for the Chatbot powered by GTP-3.5-Turbo.

## Usage

```zsh
docker compose up
```

## deploy to AWS

```zsh
docker compose up
docker compose exec serverless sls deploy --stage prod

or

docker compose run --rm serverless sls deploy --stage prod
```
